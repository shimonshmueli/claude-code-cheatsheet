# Claude Code Cheatsheet

A print-ready A3 poster cheatsheet for entry and mid-level Claude Code users, borrowing Claude's visual design language.

## Contents

- Install & launch commands
- Key concepts glossary (Session, Context window, Token, CLAUDE.md, Plan Mode, Accept Edits, MCP, Hook, Skill)
- Full slash command reference, organized by category
- Keyboard shortcuts
- @ file references & pipe/headless mode
- Permission modes (Default / Accept Edits / Plan Mode)
- Model selection guide (Sonnet 4.6 / Haiku 4.5 / Opus 4.6)
- CLAUDE.md structure & quick shortcuts
- 9 entry-level tips

## Print instructions

Open `index.html` in Chrome or Firefox, then:

1. `Cmd+P` (macOS) or `Ctrl+P` (Windows/Linux)
2. Paper size: **A3**
3. Orientation: **Portrait**
4. Margins: **None**
5. Enable **Background graphics**
6. Print or Save as PDF
